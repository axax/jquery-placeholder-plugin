Support

With this plugin you can use the HTML5 placeholder attribute in older browsers (IE7, IE8...) that don't have this feature natively. It also works with password fields. In addtion you can set the option (label2placeholder:true) to use the label as placeholder.

How to use


Javascript initialization

                        
$("input[type='text'],input[type='password']").placeholder(
{
        label2placeholder:true,
        placeholderClass:"isplaceholder"
});

Demo1

                        
<input type="text" value="" placeholder="placeholder" class="text">

Output

Demo2

                        
<div class="control-group">
    <label for="username" class="control-label ">
        Username
    </label>

    <div class="controls">
        <input type="text" value="" class="text" name="username">
    </div>
</div>
